# Sobre el equilibrio hidrostático estelar y la resolución de la ecuación de Lane-Emden
Es increíble lo mucho que sabemos sobre las estrellas, teniendo en cuenta lo lejos que están: masa, tamaño, temperatura, etc. Es gracias a la física y a las matemáticas que hemos podido estudiar las estrellas a fondo y obtener tantos datos.

Estrellas como nuestro sol están sometidas a diversos tipos de fuerzas (gravedad, electrostática, nuclear, presión), pero aun así, parecen estáticas a nuestros ojos, están en equilibrio. Esto es lo que se conoce como **equilibrio hidrostático estelar** Parece un hecho simple pero esto amplifica mucho lo que podemos deducir de las estrellas.

En este documento entraremos en ese concepto e introduciremos la **ecuación de Lane-Emden**, que nos permitirá relacionar diferentes variables para un estudio relativamente simple de las estrellas. Se soluciona esta ecuación analítica o numéricamente según el caso y se hace también un análisis del Sol con este método.
